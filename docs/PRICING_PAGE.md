**Q: What's included in the LOC limit?**  
A: Only files actively analyzed (e.g., `src/`, not `node_modules/` or `dist/`). Check your LOC with: `odavl analyze --dry-run`.

**Q: What if I exceed my LOC limit?**  
A: Analysis continues, but you'll see a warning. Upgrade to Pro for 500K LOC or Enterprise for unlimited.

**Q: Can I use ODAVL with my existing tools (ESLint, SonarQube)?**  
A: Yes! ODAVL integrates seamlessly. Import your ESLint config with `odavl import eslintrc`. Run alongside SonarQube or replace it entirely.

**Q: Do you support monorepos?**  
A: Yes. Analyze entire monorepos or specific workspaces. LOC counted per analyzed workspace.
