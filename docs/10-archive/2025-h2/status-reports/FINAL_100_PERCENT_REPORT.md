### 2. Self-Hosted (Docker)

```bash
# Clone repository
git clone https://github.com/odavl/odavl-insight.git
cd odavl-insight

# Generate Docker Compose
pnpm cli:dev devops docker compose

# Start stack
docker-compose up -d

# Access at http://localhost:3000
```

### 3. Kubernetes (AWS EKS)

```bash
# Generate Terraform + Helm
pnpm cli:dev devops terraform aws-eks
pnpm cli:dev devops kubernetes helm

# Deploy cluster
cd terraform/aws-eks
terraform apply -auto-approve

# Deploy app
aws eks update-kubeconfig --name odavl-insight-eks
helm install odavl-insight ./odavl-insight-chart

# Access via LoadBalancer
kubectl get svc odavl-insight -o jsonpath='{.status.loadBalancer.ingress[0].hostname}'
```

### 4. VS Code Extension

```bash
# Install from marketplace
code --install-extension odavl-studio.insight

# Or build from source
cd odavl-studio/insight/extension
npm install && npm run compile

# Press F5 to launch Extension Development Host
```
