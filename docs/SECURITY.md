# ODAVL Security Policy

ODAVL takes security seriously. Please follow these guidelines to report vulnerabilities.

## Reporting a Vulnerability

- Email <security@odavl.com> with details
- Do not disclose vulnerabilities publicly until resolved

## Responsible Disclosure

- We aim to respond within 48 hours
- Fixes will be prioritized and communicated

## Security Best Practices

- Keep dependencies up to date
- Use automated security scanning tools
