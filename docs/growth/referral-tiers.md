# Referral Tier System

## Reward Tiers

### Bronze Tier
**Requirements**: 0 invites  
**Rewards**:
- Access to beta program
- Early feature previews
- Community Discord access

### Silver Tier
**Requirements**: 3 successful referrals  
**Rewards**:
- 1 month Pro plan free
- Priority support
- Exclusive webinar access

### Gold Tier
**Requirements**: 10 successful referrals  
**Rewards**:
- 3 months Pro plan free
- Custom onboarding session
- Logo on partners page
- Early API access

### Platinum Tier
**Requirements**: 25 successful referrals  
**Rewards**:
- 1 year Pro plan free
- Dedicated account manager
- Co-marketing opportunity
- Lifetime community badge

## Qualification Rules

**Successful Referral**:
1. Friend signs up with your link
2. Friend activates account (runs first scan)
3. Friend remains active for 7 days

**Fraud Prevention**:
- Same IP address limited to 3 signups
- Email domain diversity required
- Manual review for 10+ referrals/day

## Reward Fulfillment

**Automatic**:
- Credits applied within 24 hours
- Email notification sent
- Dashboard badge updated

**Manual Review Required**:
- Platinum tier applications
- Bulk referrals (>20/week)
- Suspicious patterns

## Tracking

**User Dashboard Shows**:
- Total invites sent
- Successful conversions
- Current tier
- Next tier progress
- Reward history
