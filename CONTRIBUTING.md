# Contributing to ODAVL

Thank you for your interest in contributing! ODAVL welcomes contributions from everyone.

## How to Contribute

- Fork the repository and create your branch: `odavl/feature-<name>`
- Make atomic commits (≤40 LOC, ≤10 files per commit)
- Ensure all changes pass lint, build, and test
- Submit a pull request with a clear description

## Code of Conduct

All contributors must follow our [Code of Conduct](CODE_OF_CONDUCT.md).

## Reporting Issues

- Use GitHub Issues for bugs, feature requests, and questions
- Include steps to reproduce and relevant logs
