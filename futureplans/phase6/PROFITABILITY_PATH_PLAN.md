- **10 languages = complexity:** Dedicated teams per language, shared platform
- **Scale issues (1M users):** Load tested to 10M users, distributed architecture
- **Security breaches:** SOC 2, ISO 27001, penetration testing, bug bounty ($50K max)

### Market Risks
- **Competition (GitHub Copilot adds fixing):**
  - Mitigation: 76% fix rate (3-year data moat), enterprise customers (sticky)
  - Partnership: GitHub Marketplace (co-exist, don't compete)
- **Economic downturn (recession):**
  - Mitigation: ROI messaging ($100K/year saved per 100 devs)

- **Hiring 100 people fast (quality risk):**
