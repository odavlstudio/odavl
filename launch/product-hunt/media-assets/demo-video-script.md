## Production Notes

### Equipment Needed
- **Screen Recorder**: OBS Studio (free), ScreenFlow (Mac), Camtasia (paid)
- **Microphone**: USB mic (Blue Yeti) or built-in if clear
- **Editing Software**: DaVinci Resolve (free), Adobe Premiere, Final Cut Pro

### Recording Tips
- [ ] Record in 1920x1080 at 60fps
- [ ] Use demo environment (clean data)
- [ ] Close unnecessary windows/tabs
- [ ] Disable notifications
- [ ] Use cursor highlighting (if available)
- [ ] Record multiple takes (choose best)

### Voiceover Tips
- [ ] Write script verbatim, practice 3x before recording
- [ ] Speak clearly, not too fast (150-160 words/minute)
- [ ] Use enthusiastic but professional tone
- [ ] Record in quiet room (closet works!)
- [ ] Use pop filter or sock over mic
- [ ] Record multiple takes, splice best parts

### Editing Checklist
- [ ] Trim dead space at start/end
- [ ] Add smooth transitions (fade, slide)
- [ ] Sync voiceover with visuals
- [ ] Add text overlays (match brand colors)
- [ ] Include background music (royalty-free)
- [ ] Add captions (accessibility + mobile viewers)
- [ ] Export in 1920x1080 MP4 (H.264)
