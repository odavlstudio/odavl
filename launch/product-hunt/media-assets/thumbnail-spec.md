# Product Hunt Thumbnail Specification

**Size**: 1280x720 pixels (16:9 aspect ratio)  
**Format**: PNG or JPG  
**Max File Size**: 5 MB

---

## Design Requirements

### Layout
- **Background**: Dark gradient (brand-blue to brand-purple)
- **Logo**: Top-left corner, 150x150px
- **Headline**: Center, large bold text
- **Subheadline**: Below headline, smaller text
- **Visual Element**: Right side (dashboard mockup or cycle diagram)

### Typography
- **Headline Font**: Inter Bold, 72px
- **Subheadline Font**: Inter Regular, 36px
- **Color**: White text for maximum contrast

### Brand Colors
- Primary Blue: `#3B82F6`
- Primary Purple: `#8B5CF6`
- Success Green: `#10B981`
- Background Dark: `#111827`

## Visual Element Options

### Option A: Dashboard Screenshot
- Show Insight dashboard with error counts
- Highlight: "16 Detectors" and "95% Score"
- Overlay: Subtle glow effect

### Option B: O-D-A-V-L Cycle Diagram
- Circular flow: Observe → Decide → Act → Verify → Learn
- Icons for each phase
- Animated feel (motion blur on arrows)

### Option C: Before/After Split
- Left: "Before ODAVL" (red X marks, messy code)
- Right: "After ODAVL" (green checkmarks, clean dashboard)
- Dividing line in center

- **Figma**: Recommended (collaborative, templates)

- **Photoshop**: Advanced control

### Assets Needed
- ODAVL logo (SVG, 512x512)
- Brand colors palette
- Screenshot exports (PNG, high-res)
- Font files (Inter family)

---

## Checklist Before Upload

- [ ] Size exactly 1280x720 pixels
- [ ] Text readable on mobile (min 36px font)
- [ ] High contrast (dark background, light text)
- [ ] Logo visible and clear
- [ ] No typos in headline/subheadline
- [ ] File size under 5 MB
- [ ] Format PNG or JPG
- [ ] Test on multiple devices (desktop, mobile)

---

## Alternative: Video Thumbnail

If using demo video on Product Hunt:

**Requirements**:
- First frame must be visually striking
- Include "Play" button overlay
- Show product in action (not just logo)
- Keep text minimal (video will explain)

**Recommended First Frame**:
- Dashboard with live metrics updating
- Code diff showing before/after fix
- O-D-A-V-L cycle in motion
