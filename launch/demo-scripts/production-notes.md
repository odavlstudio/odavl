# Demo Video Production Guide

## Pre-Production Checklist

### Equipment
- [ ] Screen recorder (OBS Studio / ScreenFlow / Camtasia)
- [ ] USB microphone (Blue Yeti recommended)
- [ ] Pop filter or windscreen
- [ ] Quiet recording space
- [ ] Second monitor for script reference

### Software Setup
- [ ] Demo environment prepared (staging.odavl.com)
- [ ] Test data seeded (realistic but clean)
- [ ] Browser cookies cleared
- [ ] Notifications disabled
- [ ] Desktop icons hidden
- [ ] Close unnecessary applications

### Recording Settings
- [ ] Resolution: 1920x1080
- [ ] Frame rate: 60fps
- [ ] Audio: 48kHz, 16-bit
- [ ] Format: MP4 (H.264)
- [ ] Cursor highlighting enabled

## Production Tips

### Voiceover Best Practices
1. **Practice script 5x before recording**
2. **Speak at 150 words/minute** (not too fast)
3. **Use enthusiastic but professional tone**
4. **Pause 1 second between major points**
5. **Record in closet** (best acoustics)
6. **Do multiple takes** (splice best parts)
7. **Stay hydrated** (water, no caffeine)

### Screen Recording Tips
1. **Zoom browser to 90%** (more fits on screen)
2. **Move slowly** (viewers need time to read)
3. **Highlight clicks** (circle or glow effect)
4. **Rehearse cursor paths** (smooth movements)
5. **Record 2x needed length** (edit down)
6. **Separate voiceover** (easier to fix mistakes)

### Common Mistakes to Avoid
- ❌ Reading script robotically
- ❌ Moving mouse too fast
- ❌ Cluttered screen (too many windows)
- ❌ Audio peaking (check levels!)
- ❌ Forgetting to hide sensitive data
- ❌ Not testing on mobile (80% view on phone)

## Post-Production Workflow

### Editing Steps
1. **Trim dead space** (start/end)
2. **Cut mistakes** (umms, pauses, errors)
3. **Add transitions** (fade, slide - subtle!)
4. **Sync voiceover** with screen actions
5. **Add text overlays** (brand colors)
6. **Insert captions** (accessibility)
7. **Add music** (royalty-free, low volume)
8. **Color grade** (slight saturation boost)

### Text Overlay Guidelines
- **Font**: Inter Bold (brand font)
- **Size**: Minimum 48px (readable on mobile)
- **Color**: White text with dark shadow
- **Duration**: 3-5 seconds per overlay
- **Animation**: Fade in, hold, fade out
- **Content**: Key stats, feature names, CTAs

### Music Selection
- **Volume**: 20% of voiceover level
- **Style**: Upbeat tech, no vocals
- **Licensing**: Ensure commercial use allowed
- **Fade**: In at start, out at end
- **Sources**: YouTube Audio Library, Epidemic Sound

## Export Settings

### For Product Hunt
- **Format**: MP4 (H.264 codec)
- **Resolution**: 1920x1080
- **Bitrate**: 8-10 Mbps
- **Audio**: AAC, 192 kbps
- **Max Size**: 100 MB
- **Duration**: 30-90 seconds

### For YouTube
- **Format**: MP4 (H.264)
- **Resolution**: 1920x1080 or 4K
- **Bitrate**: 15-20 Mbps
- **Audio**: AAC, 320 kbps
- **No size limit**
- **Add end screen** (subscribe, links)

### For Social Media
- **Twitter**: 2:20 max, MP4, <512 MB
- **LinkedIn**: 10 min max, MP4, <5 GB
- **Instagram**: 60s max, 1080x1920 (vertical)
- **TikTok**: 3 min max, 1080x1920 (vertical)

## Quality Checklist

Before uploading, verify:
- [ ] Audio clear (no background noise)
- [ ] Video smooth (no stuttering)
- [ ] Text readable on mobile
- [ ] Captions accurate
- [ ] No sensitive data visible
- [ ] Brand logo at start/end
- [ ] CTA clear (odavl.com)
- [ ] File size within limits
- [ ] Tested on multiple devices
