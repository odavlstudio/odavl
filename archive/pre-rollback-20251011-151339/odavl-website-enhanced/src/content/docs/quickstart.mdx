# Quick Start Guide

Welcome to ODAVL! This guide will help you get started with our autonomous code quality improvement system.

## What is ODAVL?

ODAVL (Observe-Decide-Act-Verify-Learn) is an autonomous system that continuously monitors and improves code quality through automated fixes within defined safety constraints.

## Installation

```bash
npm install -g @odavl/cli
```

## Basic Usage

1. **Initialize ODAVL in your project:**
   ```bash
   odavl init
   ```

2. **Run the ODAVL cycle:**
   ```bash
   odavl run
   ```

3. **View quality reports:**
   ```bash
   odavl report
   ```

## Key Features

- **Autonomous Monitoring**: Continuous code quality assessment
- **Safe Automation**: Operates within defined safety gates
- **Learning System**: Improves decision-making over time
- **Multi-language Support**: TypeScript, JavaScript, and more
